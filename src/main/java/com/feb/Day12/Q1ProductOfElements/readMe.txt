====================================================================================================
Problem Description
----------------------------------------------------------------------------------------------------
	Write a program that returns the product of all elements present in the array.
====================================================================================================
Problem Constraints
----------------------------------------------------------------------------------------------------
	1 <= |A| <= 100
	1 <= A <= 100
	Note: It is guaranteed that the resultant product will be <= 1015
====================================================================================================	
Input Format
----------------------------------------------------------------------------------------------------
	An integer array **A** as the function argument.
	ArrayList<Integer> arr
====================================================================================================	
Output Format
----------------------------------------------------------------------------------------------------
	Product of elements in integer format
	long 
====================================================================================================
Example Input
----------------------------------------------------------------------------------------------------
	Input 1:	A = [7, 9, 2, 51]
====================================================================================================	
Example Output
----------------------------------------------------------------------------------------------------
	Output 1:	6426
====================================================================================================
Accepted Answer:
----------------------------------------------------------------------------------------------------
public static long solve(ArrayList<Integer> input) {
    // Complete the function template here
    return input.stream().mapToLong(Long::valueOf).reduce((a,b)->a*b).getAsLong();
}
