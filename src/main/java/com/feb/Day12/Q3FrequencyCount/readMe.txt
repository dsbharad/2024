====================================================================================================
Problem Description
----------------------------------------------------------------------------------------------------
	You are given an integer array A, you have to return an integer array of same size whose ith element is the frequency count of A[i] in array A .
====================================================================================================
Problem Constraints
----------------------------------------------------------------------------------------------------
	1 <= len(A) <= 1000
	1 <= A[i] <= 100
====================================================================================================	
Input Format
----------------------------------------------------------------------------------------------------
	First argument is an array of integers representing array A.
	ArrayList<Integer> A
====================================================================================================	
Output Format
----------------------------------------------------------------------------------------------------
	You have to return an array of integers as per the question.
	ArrayList<Integer>
====================================================================================================
Example Input
----------------------------------------------------------------------------------------------------
	Input 1:	A = [1, 2, 5, 1, 5, 1 ]
====================================================================================================	
Example Output
----------------------------------------------------------------------------------------------------
	Output 1:	[3, 1, 2, 3, 2, 3]
====================================================================================================
Accepted Answer:
----------------------------------------------------------------------------------------------------
import java.util.stream.IntStream;
public class Solution {
    public ArrayList<Integer> solve(ArrayList<Integer> input) {
        ArrayList<Integer> response = new ArrayList<>();
        IntStream.range(0, input.size()).forEach(index -> {
            response.add((int)input.stream().filter(item->(item.compareTo(input.get(index))==0)).count());
        });
        return response;
    }
}
