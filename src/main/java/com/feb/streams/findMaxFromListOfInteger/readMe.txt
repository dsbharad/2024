====================================================================================================
Problem Description
----------------------------------------------------------------------------------------------------
	Given a 2D ArrayList "input" of N rows and M columns. Find value of largest element in each row.
====================================================================================================
Problem Constraints
----------------------------------------------------------------------------------------------------
	1 <= N, M <= 1000
	1 <= eachElementInArrayList <= 10^9	
====================================================================================================	
Input Format
----------------------------------------------------------------------------------------------------
	ArrayList<ArrayList<Integer>> input
====================================================================================================	
Output Format
----------------------------------------------------------------------------------------------------
	ArrayList<Integer>
====================================================================================================
Example Input
----------------------------------------------------------------------------------------------------
	Input 1:	A = [[1, 2], [1, 3]]
	Input 2:	A = [[1, 2, 3]]
====================================================================================================	
Example Output
----------------------------------------------------------------------------------------------------
	Output 1:	[2, 3]
	Output 2:	[3]
====================================================================================================
