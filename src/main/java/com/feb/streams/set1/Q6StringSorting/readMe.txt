====================================================================================================
Problem Constraints
----------------------------------------------------------------------------------------------------
	Not Specified
====================================================================================================	
Input Format
----------------------------------------------------------------------------------------------------
	Original list of strings (colors): [Red, Green, Blue, Pink, Brown]

----------------------------------------------------------------------------------------------------

	Number of colors starting with 'Y': 0
====================================================================================================
Example Input
----------------------------------------------------------------------------------------------------
	Input 1:	A = [Red, Green, Blue, Pink, Brown]
====================================================================================================	
Example Output
----------------------------------------------------------------------------------------------------
	Output 1:	2
	Output 2:	0
