====================================================================================================
Problem Description
----------------------------------------------------------------------------------------------------
	Write a Java program to convert a list of strings to uppercase or lowercase using streams.
====================================================================================================
Problem Constraints
----------------------------------------------------------------------------------------------------
	Not Specified
====================================================================================================	
Input Format
----------------------------------------------------------------------------------------------------
	List of strings: [RED, grEEn, white, Orange, pink]
====================================================================================================	
Output Format
----------------------------------------------------------------------------------------------------
	Uppercase Strings: [RED, GREEN, WHITE, ORANGE, PINK]
	Lowercase Strings: [red, green, white, orange, pink]
====================================================================================================
Example Input
----------------------------------------------------------------------------------------------------
	Input 1:	A = [RED, grEEn, white, Orange, pink]
====================================================================================================	
Example Output
----------------------------------------------------------------------------------------------------
	Output 1:	[RED, GREEN, WHITE, ORANGE, PINK]
				[red, green, white, orange, pink]
